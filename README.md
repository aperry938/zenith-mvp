# ZENITH: AI Yoga Coach (Fullstack Beta)

> **"The Ghost in the Machine is now the Sage in the Browser."**

Zenith is an advanced AI Yoga Coach that uses computer vision, generative AI, and real-time feedback to guide users through their practice.

**Status:** Beta (Cycle 33)
**Architecture:** Fullstack (FastAPI + React + WebSocket)

## ðŸ— System Architecture

The application has evolved from a monolithic Streamlit prototype to a decoupled Fullstack system:

```mermaid
graph LR
    User[User / Webcam] -->|Video Stream| Client[React Frontend (Vite)]
    Client -->|WebSocket (Images)| Server[FastAPI Gateway]
    Server -->|Frame| Brain[ZenithBrain (Async Thread)]
    
    Brain -->|MediaPipe| Pose[Pose Detection]
    Brain -->|Classifier| Label[Pose Label]
    Brain -->|VAE| Quality[Form Quality Score]
    Brain -->|Flow Logic| Flow[Flow/Velocity Score]
    
    Server -->|Advice Request| Sage[Gemini Vision Client]
    Sage -->|Text Guidance| Server
    
    Brain -->|JSON Metadata| Server
    Server -->|JSON Stream| Client
    
    Client -->|Visuals| HUD[Neon HUD Overlay]
    Client -->|Audio| Voice[Web Speech API]
```

## ðŸš€ Key Features
*   **Real-Time Pose Tracking:** ultra-fast MediaPipe integration.
*   **"The Pulse" Flow Analysis:** smoothed velocity tracking to ensure fluid movement.
*   **"The Ghost" Comparison:** VAE-based reconstruction of your pose vs. ideal form.
*   **"The Sage" AI Coach:** Integration with Gemini Vision to analyze your form and give specific advice.
*   **"The Voice" Feedback:** Text-to-Speech guidance generated by the Sage.
*   **Cinematic "Retina" Visuals:** Neo-Noir aesthetic with real-time video processing.
